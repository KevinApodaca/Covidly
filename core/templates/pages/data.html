COVID Data Sources

For the Covidly Dashboard project we used data sets and data sources from the following.

  • Johns Hopkins University Center for Systems Science and Engineering
  • Our World In Data
  • New York Times

Flight Tracker Data Sources

For the Flight Tracker component of this project we used the following API data source.

  • AviationStack using the flights endpoint to capture flight departure times and arrival times for the five major Texas airports ELP, DAL, AUS, DFW, and SAT

Newsfeed Data Sources

For the newsfeed component of this project we used the following APIs and endpoints.

  • Twitter API using the recent tweets endpoint
  • NewsAPI using the top-headlines and everything endpoints.
